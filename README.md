# taxdowntest

### Dependencies
 - node v12+
 - npm v6+
 - serverless cli

### How to install
In the shell execute:
```
npm install
serverless login
```

## Development

### How to run
In the shell execute:
```
npm run dev
```
Open the brower with this url:
```
http://localhost:3000/rates
```
### How to test
In the shell execute:
```
npm run test
```
This are unit-test, and include coverage on the command line.

## Production / Cloud

### How to deploy
In the shell execute:
```
npm run deploy
```

### How to execute
Once deployed, in the shell execute:
```
npm run invoke
```
